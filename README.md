## 📌 Descripción

Proyecto académico que implementa un formulario de registro de personas con validación en el lado del cliente utilizando JavaScript moderno.

El sistema valida:

- Campos obligatorios
- Formato de correo electrónico
- Coincidencia de contraseñas
- Resaltado visual de errores
- Mensajes dinámicos mediante SweetAlert2

El objetivo es aplicar buenas prácticas de desarrollo frontend y manipulación del DOM.

## 🛠️ Tecnologías Usadas

- HTML5
- CSS3
- JavaScript (ES6+)
- SweetAlert2 (CDN)
- Visual Studio Code
- Git y GitHub

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/usuario/registro-formulario.git

cd registro-formulario

```
---


## ▶️ Uso

1. Abrir el archivo `index.html`.
2. Completar los campos:
   - Nombre
   - Correo electrónico
   - Fecha de nacimiento
   - Contraseña
   - Confirmación de contraseña
3. Presionar el botón **Registrar**.

Resultado:

- ✅ Si los datos son válidos → Se muestra mensaje de éxito.
- ⚠️ Si existen errores → Se resaltan los campos inválidos.
